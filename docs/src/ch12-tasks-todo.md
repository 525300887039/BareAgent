# 任务与 TODO

## TaskManager

<!--
- 持久化 JSON 存储（.tasks.json）
- 状态流转：pending → in_progress → done / failed
- 依赖追踪：blocks / blockedBy
- 工具：task_create, task_list, task_get, task_update
-->

## TodoManager

<!--
- 会话级内存存储，不持久化
- 优先级：high / medium / low
- 提醒机制
- 工具：todo_write, todo_read
-->

## 对应工具

<!--
| 工具 | 说明 |
|------|------|
| task_create | 创建任务 |
| task_list | 列出所有任务 |
| task_get | 获取任务详情 |
| task_update | 更新任务状态 |
| todo_write | 写入 TODO 项 |
| todo_read | 读取 TODO 列表 |
-->
